# Next Plugin for RotorHazard

A plugin that integrates RotorHazard and the Next Platform to send data in real time to pilots and the audience.
All components and full features of Next are available at https://go-next.co.

Features
Send frequencies in real-time to the Next app when the round is selected.
Upload results for each round to the Next app at the end of the round.
Send on real time data about the current lap times to Next App (Android / iOS) and Next TV for audiences.

Important!

Remember that the Next plugin for RH requires you to have the Next Local Server running on your laptop. This is the component that enables interaction between RH and Next cloud servers. If you don’t have it yet, check out this link for more information: http://go-next.co/ or file:///Users/gunthervotteler/Documents/Landing/nls.html.

Once you’ve installed the plugin and configured NLS, make sure that the pilot names match between what’s entered in RotorHazard in the 'Format' section and the pilot names in the app. It’s an easy step to verify but very important.

If you’re unsure how Next works, don’t forget to check the Wiki on YouTube.

Documentation Next Plugin.

1. Install the Next RH plugin like any other plugin. ( download de zip file, uncompress and copy inside of the folder Plugins in RotorHazard.
3. Restart RH
3. Open the Next tab in the settings page of RotorHazard (RH).
4. You can start or stop the integration service by checking or unchecking the checkbox.
5. Add the IP address of your LAN interface (desktop) that is being used to connect the lap timer and your desktop or laptop.
6. The IP address must include the port set in the NLS (Next Local Server). The default port is 5701.
7. Add the Next event id ( Copy from Next Console)
